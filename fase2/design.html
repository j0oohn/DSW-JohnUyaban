Proposito: Conceptos básicos para el diseño de sitios web

Finalidad: Aprender y familiarizarse con las etiquetas básicas de HTML, atributos y valores para el diseño de sitios web.

Tema 1 - Etiquetas básicas de HTML

Las etiquetas en HTML (Sanchez, Caronte - Custom Web & Design, 2020), son los bloques o líneas de código que se encargan de mostrar al navegador la estructura que debe tener el sitio. La estructura general de estos bloques es por los símbolos, mediante los cuales se identifica la apertura (<) y el cierre (>), el contenido que se escribe entre estos dos símbolos es la etiqueta con la instrucción para ser realizada por el navegador. A continuación se listara un breve ejemplo de varios tipos de etiquetas HTML:

Etiquetas iniciales

    html: Representa la raiz de un documento de tipo HTML

Metadatos del documento

    title: Se usa para definir el titulo del documento

Sección o estructura

    section: Define una sección del documento

Agrupación de contenido

    figure: Indica una figura ilustrada que es parte del documento

Tema 2 - Atributos y Valores

Los atributos (Sanchez, Caronte - Custom Web & Design, 2021), son valores o propiedades que se definen a una etiqueta HTML, este valor será el que determina el comportamiento de la etiqueta, de esta forma se ajustan para cumplir los criterios de los usuarios. Los atributos consisten en un nombre y un valor, separados por un signo igual (=), el valor se debe escribir en comillas dobles.
